## Key Learning Objectives

The experiments in this repository focus on understanding the **core concepts of machine learning and deep learning by implementing models from scratch**.

Main goals include:

- Understanding the **Perceptron algorithm**
- Implementing **Logistic Regression for classification**
- Building **Linear and Multiple Linear Regression models**
- Implementing **Gradient Descent manually**
- Learning how **activation functions introduce non-linearity**
- Understanding **hidden layers and neural network architecture**
- Studying **model capacity and overfitting**
- Exploring how **neural networks learn complex patterns**
